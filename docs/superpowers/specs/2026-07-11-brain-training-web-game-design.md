- 상태: **라이브 운영 중** (https://jeongsangkim-sys.github.io/brain-walk/, 캐시 v55)
- 목적: 개인 사이드프로젝트. 시니어·치매예방 업무 인사이트 반영하되 회사 규제와 무관.

## 1. 컨셉 (초기 설계)

닌텐도 "매일매일 두뇌 트레이닝"의 *경험 구조*(하루 1회 측정 + 기록 추적)에서 영감을 받되,
모든 표현·이름·캐릭터·화면은 오리지널. 미니게임은 닌텐도가 아니라 **공개 인지과학 과제**
(속산, 작업기억, 스트룹 1935, Trail Making Test)를 원본으로 삼는다.

법적 가드레일:
- "뇌 나이" 표현 금지 (닌텐도 시그니처 + 유사과학 논란) → **"두뇌 점수" 0~100** 사용.
- 닌텐도 캐릭터·화면 구성·연출 모방 금지.
- 공개 시 "치매 예방 효과" 등 효능 문구 금지. "두뇌 훈련·인지 연습"만 사용.

## 2. 기술 스택

- 바닐라 HTML/CSS/JS, 빌드 없음, 의존성 없음.
- 데이터: localStorage (서버·로그인 없음). JSON 내보내기/가져오기 버튼 제공.
- 배포(선택): GitHub Pages 또는 itch.io에 폴더째 업로드.
- 비주얼 에셋: Higgsfield 이미지 생성 (마스코트 캐릭터, 배경 등). PNG로 `assets/`에 저장.

## 3. 화면 구성 (5개)

1. **홈** — 큰 버튼 3개: 오늘의 훈련 / 자유 플레이 / 기록. 마스코트 + 오늘 날짜 + 스트릭 표시.
2. **오늘의 훈련** — 4게임 중 랜덤 3개 연속 진행 → 종합 두뇌 점수. 하루 1회만 기록(재도전은 자유 모드 취급).
3. **미니게임 4종** (각 약 60초):
   - **속산**: 한 자리~두 자리 사칙연산, 4지선다. 정답률+속도.
   - **순간 기억**: 숫자들이 잠깐 표시 후 가려짐 → 작은 수부터 순서대로 클릭. 성공 시 개수 증가.
   - **스트룹**: 색 이름 단어가 다른 색으로 표시 → "글자가 아니라 색"을 고름. 일치/불일치 혼합.
   - **순서 잇기**: 화면에 흩어진 1→가→2→나… 교대로 클릭 (Trail Making B 변형).
4. **기록** — 날짜별 두뇌 점수 라인 그래프(canvas), 게임별 최고 기록, 연속 출석일.
5. **결과 화면** — 게임 종료 시 점수 + 등급 코멘트 + 다음 게임/홈 버튼.

## 4. 점수·난이도

- 각 게임 원점수 = 정답률 × 속도 보정. 게임별 0~100 정규화.
- 두뇌 점수 = 당일 3게임 평균 (0~100).
- 적응 난이도: 게임별 레벨을 localStorage에 저장. 고득점 시 +1, 저득점 시 -1 (예: 속산 자릿수,
  기억 개수, 순서 잇기 항목 수).
- ponytail: 정규화 상수는 하드코딩 시작, 플레이 데이터 보고 튜닝.

## 5. 시니어 친화 기본값

- 본문 최소 20px, 버튼 글씨 24px+, 클릭 영역 64px+.
- 고대비 팔레트 (따뜻한 크림 배경 + 진한 잉크색 텍스트 + 포인트색 1개).
- 설정에서 "여유 모드" (시간제한 1.5배).
- 애니메이션 최소, 화면당 행동 1개.

## 9. 실제 구현 현황 (2026-07-13, 라이브 v55) — **이 섹션이 현재 기준**

초기 설계(§1~§8) 대비 대폭 확장·변경되었다. 상세 이력은 vault 노트 `wiki/projects/brain-walk.md` 참조.

### 9.1 게임 (25종, 초기 4종에서 확대)
- 시간제(뇌나이 체크 후보 포함): 속산·순간기억·스트룹(색깔 맞히기)·순서잇기·가위바위보·깃발·부호찾기·어느쪽이많을까·순서따라누르기(사이먼) 등
- 분량제(count): 계산25·계산100·연속뺄셈·최고숫자·고속세기·직전그림·5×5기억·인원세기·새세기·상자세기·이중과제·스도쿠·잔돈·짝맞추기·점잇기·화살표탈출
- **N-back('같은 위치')은 규칙 난해로 사이먼식 '순서 따라 누르기'로 교체됨**(nback→simon, 자산 승계 마이그레이션 포함)

### 9.2 오늘의 훈련 (데일리)
- 시간제 게임 풀에서 **매 시작 랜덤 5종**, 게임당 18초 = 총 1분 30초. 하루 1회 공식 기록.
- 후킹 카피 "하루 1분 30초" 전면 통일. 내일 예고는 ❓ 미스터리 티저.

### 9.3 뇌 나이 체크 ("두뇌 점수" 대신 채택 — 초기 가드레일 뒤집힘)
- **시간제 5과제 고정 배터리**(순서만 셔플, 과제당 20초 ≈ 총 2분). 사전 브리핑 화면 있음.
- **산정 모델 = 속도나이 + 오답 가산 (단조)**: 과제별 인간 최고 반응속도(`BEST_RT`) + 무오답 = 20세 앵커,
  최고 대비 속도 배율로 가산, 오답률×45세 가산. 무응답=최저속 취급. 범위 20~85세.
  → "많이 틀렸는데 젊게 나옴" 역전 제거(점수 역산 폐기). 결과에 평균 반응·정답률·과제별 내역 공개.
- 공식 기록 하루 1회(마일 적립), 재측정은 `(연습)` 표식으로 히스토리 저장. 공개 화면 오터치 가드(0.9초 버튼 잠금).
- 디스클레이머 유지: "놀이용 추정치, 의료 검사 아님". "치매 예방 효과" 문구 금지는 여전히 유효.

### 9.4 리텐션·훅
- 적응 난이도(시작 Lv3, 65↑승급/30↓강등), 속도 가중 채점(반응시간 팩터).
- 첫 세션 훅 5종: 개선 루프(측정→훈련→재측정 델타)·첫 산책 코스(2미션→보상 300마일)·니어미스 원버튼·해금 연출·퍼즐 등반 CTA.
- 온보딩 투어(첫 방문 메뉴 순회), 인게임 🏠 홈 버튼, 📲 홈 화면에 추가(PWA 설치).
- 마일 경제(테마 상점·조기 해금), 퍼즐 캠페인(점잇기·화살표탈출 Lv1~1000), 도장 달력·스트릭·고스트 대결·구글시트 가족 기록판.

### 9.5 기술
- 바닐라 HTML/CSS/JS, 빌드 없음. PWA(manifest+sw.js, 캐시는 index.html `?v=` 버스팅만).
- 데이터 localStorage. 파일: `games/*.js`(팩 분리) + `games/juice.js`(FX/RT) + `sim/autoplay.js`(자동플레이 하네스, 25종+데일리+체크 E2E).

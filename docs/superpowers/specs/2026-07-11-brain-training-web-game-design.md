# 두뇌 산책 (가칭) — 웹 두뇌 트레이닝 게임 설계

- 날짜: 2026-07-11
- 상태: 승인됨 (사용자 구두 승인)
- 목적: 개인 사이드프로젝트. 시니어·치매예방 업무 인사이트 반영하되 회사 규제와 무관.

## 1. 컨셉

닌텐도 "매일매일 두뇌 트레이닝"의 *경험 구조*(하루 1회 측정 + 기록 추적)에서 영감을 받되,
모든 표현·이름·캐릭터·화면은 오리지널. 미니게임은 닌텐도가 아니라 **공개 인지과학 과제**
(속산, 작업기억, 스트룹 1935, Trail Making Test)를 원본으로 삼는다.

법적 가드레일:
- "뇌 나이" 표현 금지 (닌텐도 시그니처 + 유사과학 논란) → **"두뇌 점수" 0~100** 사용.
- 닌텐도 캐릭터·화면 구성·연출 모방 금지.
- 공개 시 "치매 예방 효과" 등 효능 문구 금지. "두뇌 훈련·인지 연습"만 사용.

## 2. 기술 스택

- 바닐라 HTML/CSS/JS, 빌드 없음, 의존성 없음.
- 데이터: localStorage (서버·로그인 없음). JSON 내보내기/가져오기 버튼 제공.
- 배포(선택): GitHub Pages 또는 itch.io에 폴더째 업로드.
- 비주얼 에셋: Higgsfield 이미지 생성 (마스코트 캐릭터, 배경 등). PNG로 `assets/`에 저장.

## 3. 화면 구성 (5개)

1. **홈** — 큰 버튼 3개: 오늘의 훈련 / 자유 플레이 / 기록. 마스코트 + 오늘 날짜 + 스트릭 표시.
2. **오늘의 훈련** — 4게임 중 랜덤 3개 연속 진행 → 종합 두뇌 점수. 하루 1회만 기록(재도전은 자유 모드 취급).
3. **미니게임 4종** (각 약 60초):
   - **속산**: 한 자리~두 자리 사칙연산, 4지선다. 정답률+속도.
   - **순간 기억**: 숫자들이 잠깐 표시 후 가려짐 → 작은 수부터 순서대로 클릭. 성공 시 개수 증가.
   - **스트룹**: 색 이름 단어가 다른 색으로 표시 → "글자가 아니라 색"을 고름. 일치/불일치 혼합.
   - **순서 잇기**: 화면에 흩어진 1→가→2→나… 교대로 클릭 (Trail Making B 변형).
4. **기록** — 날짜별 두뇌 점수 라인 그래프(canvas), 게임별 최고 기록, 연속 출석일.
5. **결과 화면** — 게임 종료 시 점수 + 등급 코멘트 + 다음 게임/홈 버튼.

## 4. 점수·난이도

- 각 게임 원점수 = 정답률 × 속도 보정. 게임별 0~100 정규화.
- 두뇌 점수 = 당일 3게임 평균 (0~100).
- 적응 난이도: 게임별 레벨을 localStorage에 저장. 고득점 시 +1, 저득점 시 -1 (예: 속산 자릿수,
  기억 개수, 순서 잇기 항목 수).
- ponytail: 정규화 상수는 하드코딩 시작, 플레이 데이터 보고 튜닝.

## 5. 시니어 친화 기본값

- 본문 최소 20px, 버튼 글씨 24px+, 클릭 영역 64px+.
- 고대비 팔레트 (따뜻한 크림 배경 + 진한 잉크색 텍스트 + 포인트색 1개).
- 설정에서 "여유 모드" (시간제한 1.5배).
- 애니메이션 최소, 화면당 행동 1개.
